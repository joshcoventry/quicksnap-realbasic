QuickSnap Help Index:Solving problems

Solving problems

Did you encounter an issue with QuickSnap? The following answers may be helpful. If there is no solution to your problem here, e-mail us at quicksnap@fastforwardsw.com and we'll get back to you.

I took a screenshot but the file doesn't show up...
Make sure you are looking for the file in the correct place. Check the destination for screenshot files in the 'General' section of the Preferences window. If you still can't find your screenshot file, it might not have been created in the first place due to a bug or malfunction in your preferences. Deleting the Preferences file found out <Your Home Folder>/Library/Preferences/QuickSnap Prefs might solve the issue; however after deleting this file you will have to re-enter your license information if you are a registered user of QuickSnap.
I have found a bug!
If are experiencing abnormal behavior while using QuickSnap, please report it to us so we can look into it! It helps tremendously if you are able to reproduce by taking the same steps each time.

If your bug seems to be related to a specific set of preferences or options you are using, please send us your preferences file located at <Your Home Folder>/Library/Preferences/QuickSnap Prefs
QuickSnap crashed!

Please e-mail us about any crashes you encounter. We fix all known crashes as soon as possible, but if we don't know it exists it might stick around forever.

A source of vital information about crashes is the crash log file your computer generates. Please include this file* in your e-mail to us. Try to be as detailed as possible about what you were doing when QuickSnap crashed. It will help us remove these problems as fast as we can.

* located at <Home Folder>/Library/Logs/CrashReporter/QuickSnap.crash.log

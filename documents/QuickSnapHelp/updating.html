QuickSnap Help Index:Updating

Updating

QuickSnap's built-in software updater can inform you when new updates are available. An internet connection is required for this feature of QuickSnap.
Checking for updates
  1. Open the Preferences window by clicking the Options button in the main window.
  2. Click the Software Update button.
  3. Click the Check Now button.
A QuickSnap update is available...what do I have to do?
If QuickSnap has informed you of an update, download it from our website. Once the disk image has been downloaded, it should automatically mount onto your desktop and open. Make sure your current version of QuickSnap is closed and not actively running. Drag the new QuickSnap application file from the disk image to the location of your current version of QuickSnap. When asked if you want to replace it, click Replace. You have now succesfully updated QuickSnap, and can continue using it as usual.
Checking for updates automatically at startup
You can choose to let QuickSnap inform you of new updates automatically when you open QuickSnap by checking the checkbox in the Preferences window for this option.
